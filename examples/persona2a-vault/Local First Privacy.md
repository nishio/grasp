# Local First Privacy

Local-first is a hook, not the whole product.

The vault stays as Markdown. `grasp` builds a local SQLite graph beside it, so an agent can inspect [[Agent Memory]] without sending the whole folder to a hosted notes app.

Privacy alone does not answer why this is better than grep. The stronger claim is bounded graph retrieval: [[Context Budget]] plus provenance and write guards.
